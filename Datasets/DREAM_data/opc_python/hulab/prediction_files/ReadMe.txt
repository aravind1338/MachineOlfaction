Description of files:

hulab_preprocessing.py			:	performs data preprocessing
hulab_prediction.py			:	performs prediction for the 2 subchallenges

The unziped data from hulab_data.zip (features, weights etc.) must be in the same folder as the python scripts. Scripts in hulab_preprocessing.py must be executed first, before hulab_prediction.py.

hulab_data.zip contains some of the original data so it is not included, available in our submitted folder in synapse.
